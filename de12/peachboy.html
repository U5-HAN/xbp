鬼退治orキャバ対峙！生きるか死ぬかの桃太郎デスゲーム

【ゲーム説明】


あなた（＝主人公）は桃太郎です。桃太郎が鬼退治もしくはキャバ嬢と対峙することで、生きるか死ぬかのデスゲームが始まります。
ちなみにただのシミュレーションゲームなので全然難しくありません。ただ、死んだらそこでゲーム終了なので頑張って最後まで突き進んでください。

【ソースコード】


         
            import time

# ゲーム紹介
def introduce_games():
    print("ぱっかーん！あなたは桃から生まれた桃太郎。早速世界平和のためにどこへ何をしに行くか決めよう！")
    print("1. 鬼ヶ島へ、鬼退治に行くぜ！")
    print("2. キャバクラで、キャバ嬢と対峙しにいくぜ！")

# ゲームスタート
def start_game():
    print("鬼退治orキャバ対峙！生きるか死ぬかの桃太郎デスゲーム開幕！")
    introduce_games()
    time.sleep(1)

    print("\nどちらに向かいますか？")
    choice = input("1. 鬼ヶ島\n2. キャバクラ\n選択してください（1または2）: ")

    if choice == "1":
        go_to_onigashima()
    elif choice == "2":
        go_to_kyabakura()
    else:
        print("無効な入力です。もう一度お試しください。")
        start_game()
